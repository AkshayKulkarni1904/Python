## 📌 About This Repository

This is my **personal Python learning journal** — a work-in-progress repository where I track daily practice sessions, notes, and exercises as I build my Python skills from scratch.

> **Goal:** Master Python fundamentals and progress into data science (NumPy, Pandas, Matplotlib) through consistent daily practice.

## 🚀 How to Run Notebooks

### Option 1 — Google Colab (Recommended, Zero Setup)

Each notebook has an **"Open in Colab"** badge at the top. Click it to run instantly in your browser.

### Option 2 — Run Locally

**Prerequisites:** Python 3.10+ and `pip` installed.

```bash
# 1. Clone this repository
git clone https://github.com/AkshayKulkarni1904/Python.git
cd Python

# 2. (Optional) Create a virtual environment
python -m venv venv
source venv/bin/activate       # macOS/Linux
venv\Scripts\activate          # Windows

# 3. Install dependencies
pip install jupyter numpy pandas matplotlib seaborn

# 4. Launch Jupyter
jupyter notebook
```

Then navigate to the `notebooks/` folder and open any `.ipynb` file.
